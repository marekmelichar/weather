##Installation

```bash
    $ npm install
```

##Usage

To use webpack dev server & watch,

```bash
    $ npm start
```

To build on production mode

```bash
    $ npm run build
```

To run some tests I would recommend Mocha, Chai, jQuery-Chai

```bash
    $ npm run test
```

Listens at **http://localhost:8082**
